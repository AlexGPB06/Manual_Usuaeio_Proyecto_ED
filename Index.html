Manual de Usuario: Banco Financiero

Guía interactiva para operar en nuestra plataforma de consola.

Al iniciar el programa, se te presentará el menú principal. Desde aquí puedes acceder con una cuenta existente o crear una nueva.

Para Registrar un Nuevo Cliente: Elige la opción '2'. El sistema te pedirá tus datos personales y generará automáticamente un ID y un número de tarjeta únicos para ti.


  Seleccione una opción: 2

╭──────────────────────────────────╮
│         REGISTRO DE CLIENTE      │
╰──────────────────────────────────╯
  > Ingrese su Nombre: Maria
  > Ingrese su Apellido: Lopez
  > Ingrese un monto inicial: 1500

   ¡Registro exitoso!
   - Cliente: Maria Lopez
   - ID asignado: 3
   - Nº Tarjeta: 4509881234567890
   - Saldo Inicial: $1,500

Presione Enter para continuar...

Para Iniciar Sesión: Elige la opción '1' e introduce las credenciales que usaste durante el registro. El sistema validará cada dato para garantizar un acceso seguro.


  Seleccione una opción: 1

╭──────────────────────────────────╮
│          INICIO DE SESIÓN        │
╰──────────────────────────────────╯
  > Nº Tarjeta (16 dígitos): 5201169781530257
  > ID de Cliente: 1
  > Nombre: Juan
  > Apellido: Perez

   ¡Inicio de sesión exitoso! Bienvenido, Juan Perez

Presione Enter para continuar...

Una vez dentro, tu panel principal te mostrará un resumen claro de tu cuenta y una lista de las operaciones disponibles. Cada opción está numerada para facilitar la navegación.

  • Opciones 1-3: Operaciones transaccionales básicas.
  • Opciones 4-5: Consultas sobre tu actividad financiera.
  • Opción 6: Herramienta de ahorro a largo plazo.

   ╔══════════════════════════════╗
   ║        BANCO FINANCIERO      ║
   ╠══════════════════════════════╣
   ║ Cliente: Juan Perez         
   ║ ID Cliente: 1                 
   ║ Saldo: $7,500                 
   ╠══════════════════════════════╣
   ║ 1. Realizar deposito         ║
   ║ 2. Retirar monto             ║
   ║ 3. Realizar transferencia    ║
   ║ 4. Ver movimientos           ║
   ║ 5. Filtrar movimientos x monto ║
   ║ 6. Ir a Caja de Ahorros      ║
   ║ 7. Cerrar sesión             ║
   ╚══════════════════════════════╝
   > Seleccione opcion:

Realiza tus operaciones diarias de forma sencilla. El sistema te guiará en cada paso y actualizará tu saldo automáticamente.

Ejemplo de Depósito:


   > Seleccione opcion: 1
   > Monto a depositar: 1000
   > Saldo anterior: $7,500
   > Saldo nuevo:    $8,500
   > Depósito realizado con éxito.

Presione Enter para continuar...

Ejemplo de Retiro:


   > Seleccione opcion: 2
   > Monto a retirar: 300
   > Saldo anterior: $8,500
   > Saldo nuevo:    $8,200
   > Retiro realizado con éxito.

Presione Enter para continuar...

Ejemplo de Transferencia: Para máxima seguridad, el sistema te pedirá validar la identidad del destinatario con su Tarjeta y su ID antes de confirmar la transacción.


   > Seleccione opcion: 3

   ╔══════════════════════════════╗
   ║        TRANSFERENCIA         ║
   ╠══════════════════════════════╣
   ║ > Nº Tarjeta destinatario: 4509297861614535
   ║ > ID destinatario: 2
   ║ > Monto a transferir: 500
   ║ ¡Transferencia realizada con éxito! ║
   ╚══════════════════════════════╝

Presione Enter para continuar...

Puedes consultar tu historial de dos formas. La vista estándar te muestra las transacciones más recientes primero, gracias a que se almacenan en una Pila (Stack).

Historial Estándar (LIFO - Último en entrar, primero en salir):


   > Seleccione una opción: 4

   ╠══════════════════════════════╣
   ║     HISTORIAL DE MOVIMIENTOS     ║
   ╠══════════════════════════════╣
   ║ > Transferencia: $-500         ║
   ║ > Retiro: $-300                ║
   ║ > Depósito: $1000              ║
   ╚══════════════════════════════╝

Presione Enter para continuar...

Si necesitas analizar tus gastos, la opción "Filtrar por monto" utiliza un Árbol Binario para ordenar eficientemente tus transacciones, mostrando primero las salidas de dinero (negativos) de menor a mayor.

Movimientos ordenados por Monto:


   > Seleccione una opción: 5

   ╠══════════════════════════════╣
   ║ Movimientos ordenados por monto: 
   ║ $-500
   ║ $-300
   ║ $1000
   ╚══════════════════════════════╝

Presione Enter para continuar...

La Caja de Ahorros es una cuenta separada para ayudarte a cumplir tus metas. Tiene su propio menú, donde el saldo máximo es de $500,000.

Puedes calcular el crecimiento de tu dinero a futuro. Este cálculo se realiza con un algoritmo recursivo que proyecta el interés compuesto mes a mes.


   ╔══════════════════════════════╗
   ║         CAJA DE AHORROS      ║
   ╠══════════════════════════════╣
   ║ Saldo Principal: $7,700        ║
   ║ Saldo Ahorrado:  $0            ║
   ╠══════════════════════════════╣
   ║ 1. Depositar en Ahorros      ║
   ║ 2. Retirar de Ahorros        ║
   ║ 3. Ver Proyección de Ahorros ║
   ║ 4. Volver al menú principal  ║
   ╚══════════════════════════════╝

   > Monto a depositar (entero, mín. $100): 5000
